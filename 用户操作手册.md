# 瑞界物业承接查验系统 — 用户操作手册

## 系统概述

本系统用于物业承接查验的数据录入、存档和打印。核心使用场景：

> 创建项目 → 从模板库选表保存为项目表单 → 打印空白表去现场 → 纸质填写 → 回系统录入 → 签字归档

系统有三种角色：**管理员**（全部权限）、**项目经理**（管理项目、表单、成员）、**普通员工**（录入检查结果）。

---

## 一、承接查验完整流程

### 第 1 步：管理员创建项目

1. 登录后进入项目列表，点击「+ 新建项目」
2. 填写项目名称（如"北宁嘉平工业园一期"），系统自动生成编号（P0001）
3. 可选填项目类型、地址、面积、交接日期等
4. 点击「创建」

### 第 2 步：项目经理配置楼栋和房源

1. 进入项目 → 右上角「配置」
2. 「楼栋管理」标签：输入楼栋名称（如 1栋、A座）→ 点击「添加」
3. 「房源管理」标签：选择楼栋 → 输入房号（如 101）→ 点击「添加」

### 第 3 步：从参考表单保存为项目表单

1. 左侧点击「📁 参考表单」
2. 浏览 27 套模板（按 A设备机房 / B公共区域 / C室内 / D资料移交 分类）
3. 点击目标模板 → 进入模板详情页
4. 可双击编辑检查项、添加或删除条目
5. 点击「保存为我的表单」→ 表单名自动生成：**模板标题 - 用户名**

> **典型承接查验选表清单（15-20 张）：**
> - A1 变配电室、A2 发电机房、A3 水泵房、A6 冷水机组
> - A7 通风排烟、A8 消防水系统、A9 消防报警
> - A5 电梯机房、A10 电梯轿厢及井道
> - B1 监控安防、B2 道路停车场、B4 围墙大门
> - B6 给排水外网、B7 弱电智能化、B8 防雷接地
> - C1 室内土建、C2 室内给排水、C3 室内电气
> - D1 技术资料移交、D2 钥匙门禁移交、D4 问题汇总

### 第 4 步：打印空白表，带去现场

1. 进入「📋 我的表单」→ 点击目标表单的「🖨 打印空白表」
2. 跳转到打印预览页 → 点击「下载 PDF」或「浏览器打印」
3. 打印的纸质表包含：表单标题、项目名称、检查项目和标准、空白结果列
4. ≤10 项的表单自动单页

### 第 5 步：现场查验，纸质填写

1. 携带空白表到对应设备/区域
2. 逐项检查，在纸上勾选：合格 / 不合格 / 免检
3. 不合格项注明问题描述
4. 拍摄现场照片（供后续录入时上传）
5. 填写查验意见、查验人签字、日期

### 第 6 步：回系统录入查验结果

1. 进入「📝 检查记录」→ 点击「+ 新增记录」
2. 选择对应表单
3. 系统进入查验录入模式（标题旁显示红色「查验录入」标识）
4. 补充楼栋、房源、位置信息
5. 逐项点击「合格」「不合格」「免检」—— 结果自动保存
6. 不合格项填写问题描述
7. 点击「📷」按钮上传现场照片（每项最多 6 张）
8. 填写「查验意见」→ 状态改为「已完成」
9. 点击「保存记录」

### 第 7 步：打印签字归档

1. 「📝 检查记录」列表 → 点击已完成记录 → 打印
2. 带结果的 PDF 包含：完整检查结果、问题描述、查验意见、签字栏
3. 打印、签字、盖章 → 归档

## 二、数据安全规则

| 规则 | 说明 |
|---|---|
| **已完成锁定** | 状态改为"已完成"后，前后端均不可再修改 |
| **记录归属** | 普通员工只能修改自己创建的记录；管理员可改所有 |
| **数据快照** | 记录创建时检查项名称、标准当场固定，后续模板修改不影响历史记录 |
| **每次 = 新记录** | 同一表单每次新建记录，互不覆盖 |

## 三、承接查验完成后，日常巡查怎么用

同一张表单（如"水泵房巡查表"），每次巡查从「检查记录」→「+ 新增记录」→ 选同一表单 → 新建一条新记录。

筛选表单后，列表按时间倒排，所有历史记录一目了然：

```
📝 检查记录                    [水泵房巡查表 ▼] [全部楼栋 ▼] [全部状态 ▼]

  水泵房巡查表 · 李电工 · 1号水泵房 · 2026-05-23 · 16项 · 已完成
  水泵房巡查表 · 张电工 · 2号水泵房 · 2026-05-19 · 12项 · 查验中
  水泵房巡查表 · 张电工 · 1号水泵房 · 2026-05-12 · 16项 · 已完成
  ...
```

承接查验的历史记录永久保留，和后续日常巡查记录并列，形成设备完整履历。

## 六、常见问题

**Q: 表单太多找起来麻烦？**
A: 在"参考表单"页按 ABCD 分类浏览；在"检查记录"页可按表单、楼栋、状态筛选。

**Q: 现场没有打印机？**
A: 提前在办公室打印好所有空白表；也支持手机浏览器直接访问系统录入。

**Q: 检查时发现模板里没有的项目？**
A: 录入模式下点击「+ 添加条目」可临时增加检查项。

**Q: 录错了怎么办？**
A: 状态为"查验中"时随时可改。改为"已完成"前请确认——完成后锁定不可改。

**Q: 承接查验做完，日常巡查怎么用？**
A: 每次巡查从「检查记录」→「+ 新增记录」→ 选同一表单 → 新建记录。历史承接记录永久保留。

**Q: 如何备份数据？**
A: 数据库文件位于 `backend/data.db`，定期复制即可。
