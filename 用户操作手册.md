## 1. 系统概述

瑞界物业管理系统是一套面向物业承接查验、日常巡检和设备维护的综合管理工具。系统采用单项目设计，一个实例管理一个物业项目。

### 核心功能模块

| 模块 | 说明 | 适用角色 |
|------|------|----------|
| 我的表单 | 创建和管理自用的查验表单 | 全员 |
| 日常巡检 | 日常巡查记录的填写、保存、提交 | 全员 |
| 承接查验 | 新项目承接查验记录的录入与管理 | 全员 |
| 设备档案 | 设备台账、维护计划、说明书管理 | 全员查看，经理编辑 |
| 参考表单 | 系统预设的标准化查验模板库 | 全员查看，管理员编辑 |
| 打印 | 空白表和已填记录的 A4 打印输出 | 全员 |
| 仪表盘 | 统计概览、趋势图表、人员完成率 | 经理/管理员 |
| 项目配置 | 项目信息、楼栋房源、成员管理 | 经理/管理员 |

## 2. 登录与账户

### 2.1 登录

1. 打开系统网址，进入登录页面
2. 输入用户名和密码
3. 点击"登录"或按 Enter 键
4. 登录成功后自动进入项目首页

### 2.2 修改密码

1. 点击左侧菜单底部的"修改密码"
2. 在弹出的对话框中输入旧密码、新密码（至少 4 位）、确认新密码
3. 点击"确认"

### 2.3 退出登录

点击左侧菜单底部的"用户名 · 退出"。

## 3. 角色与权限

系统设有三个角色，权限由低到高：

### 基层员工（employee）

- 创建和编辑自己的表单
- 填写和提交查验记录
- 查看设备档案和参考表单
- 不可访问仪表盘、项目配置、用户管理
- 不可修改或删除他人创建的记录

### 物业经理（manager）

- 包含员工的所有权限
- 访问仪表盘，查看项目统计
- 访问项目配置，管理楼栋、房源、成员
- 添加和编辑设备信息
- 不可访问用户管理（管理员专属）

### 系统管理员（admin）

- 包含经理的所有权限
- 访问用户管理，创建和删除系统用户
- 编辑参考表单模板

### 记录提交与锁定

| 操作 | 创建者 | 经理/管理员 |
|------|:------:|:----------:|
| 保存草稿 | ✓ | ✓ |
| 提交锁定 | ✓ | ✓ |
| 查看已提交记录 | 仅自己 | 全部 |
| 修改已提交记录 | ✗ | ✗ |
| 删除已提交记录 | ✗ | ✗ |
| 修改他人记录 | ✗ | ✗ |

> **重要**：记录一旦"提交"即永久锁定，任何人无法再修改或删除。提交前请仔细核对数据。

## 4. 我的表单

**路径**：左侧菜单 → 工作台 → 我的表单

"我的表单"是每位用户的表单管理中心，也是登录后的默认首页。

### 4.1 创建空白表单

1. 点击"创建空白表单"按钮
2. 输入表单标题（如"1号楼配电房巡检表"）
3. 点击确认，进入表单编辑器

### 4.2 从参考模板创建

1. 进入"参考表单"页面
2. 找到需要的模板，点击"查看"
3. 在模板详情页点击"保存为我的表单"
4. 系统自动复制模板内容为你的个人表单

### 4.3 编辑表单

1. 在"我的表单"列表中找到目标表单
2. 点击"编辑"，进入表单编辑器
3. 可进行的操作：
   - 修改标题：直接在顶部标题处编辑
   - 添加条目：点击"添加条目"按钮，输入检查项目名称和检查标准
   - 编辑条目：双击条目名称或检查标准进行修改
   - 删除条目：点击条目右侧的删除按钮
   - 批量编辑：点击"批量编辑"按钮，所有条目进入编辑模式

### 4.4 删除表单

点击表单卡片的"删除"按钮，确认后删除。

> 注意：删除表单会同时删除该表单下的所有查验记录，请谨慎操作。

## 5. 日常巡检

**路径**：左侧菜单 → 工作台 → 日常巡检

日常巡检用于物业日常巡查工作的记录，如配电房巡检、消防设施检查等。

### 5.1 新建巡检记录

1. 点击"新增记录"按钮
2. 在弹出的对话框中选择一个你的表单
3. 系统自动跳转到查验录入页面

### 5.2 填写查验记录

#### 选择位置

- 楼栋：从下拉列表中选择
- 房号：选择楼栋后自动过滤该楼栋的房源
- 位置补充：可自由填写更具体的位置信息（如"3层配电间"）

#### 逐项检查

每个检查项有三个选项：

| 结果 | 含义 | 操作 |
|------|------|------|
| 合格 | 符合检查标准，无问题 | 直接点击 |
| 不合格 | 不符合标准，存在缺陷 | 点击后需填写问题描述 |
| 免检 | 本次不检查此项 | 直接点击（如不适用或无法检查） |

选择结果后自动保存，无需手动点击保存按钮。

#### 拍照留证

每个检查项可以上传照片：

1. 点击某项的相机按钮（显示已上传数量）
2. 在照片弹窗中点击上传区域添加照片
3. 每项最多 6 张照片，每张不超过 10MB
4. 支持手机拍照或从相册选择

#### 填写意见

在页面底部的"查验意见"文本框中输入整体评价或备注。

### 5.3 保存与提交

- 保存记录：保存当前位置、意见等元数据，记录保持可编辑状态
- 提交：不可逆操作，提交后记录永久锁定

点击"提交"时会弹出确认对话框，确认后记录变为只读，顶部显示"已提交"徽标。

## 6. 承接查验

**路径**：左侧菜单 → 工作台 → 承接查验

承接查验与新项目交付验收相关，操作流程与日常巡检完全一致，区别在于：

- 承接查验使用 `record_type: acceptance` 标记
- 在列表中与日常巡检分开展示
- 通常使用资料移交、房屋验收等特定模板

## 7. 设备档案

**路径**：左侧菜单 → 工作台 → 设备档案

设备档案管理物业项目内的所有设备台账、维护计划和说明书。

### 7.1 设备列表

以卡片网格展示所有设备。每张卡片包含：

- 维护状态指示灯：绿 7天后到期 / 黄 7天内到期 / 红 已逾期
- 设备名称、型号、编号
- 安装位置（楼栋-房号-具体位置）
- 设备状态徽标

筛选功能：
- 按类别筛选：配电 / 消防 / 电梯 / 给排水 / 暖通 / 弱电 / 其他
- 按状态筛选：正常 / 保养中 / 待修 / 已报废

### 7.2 添加设备（经理/管理员）

1. 点击"添加设备"按钮
2. 填写设备信息：设备名称（必填）、类别、状态、型号、编号、安装日期、安装位置、备注
3. 点击确认保存

### 7.3 编辑/删除设备

- 点击设备卡片的"编辑"按钮修改信息
- 点击"删除"按钮删除设备（需确认）

### 7.4 设备详情

点击设备卡片的"查看"进入详情页，包含三个标签页：

#### 基本信息

设备的所有属性可编辑（经理/管理员），支持修改并保存。

#### 维护计划

记录每次维护的详细信息：

- 新增维护计划：设置计划日期和维护内容
- 维护记录表格：显示计划日期、内容、状态、完成日期、备注、照片
- 状态管理：
  - 待执行（蓝色）— 尚未执行
  - 已完成（绿色）— 已按期完成
  - 已跳过（灰色）— 本周期跳过
  - 已逾期（红色）— 超期未执行
- 编辑记录：点击某条记录，在弹窗中更新状态、完成日期、备注，上传维护照片
- 提交记录：维护完成后可提交锁定
- 统计行：显示"X 项 · Y 已完成 · Z 待执行"

#### 说明书

上传和管理设备说明书、操作手册等文件：

- 支持 PDF、Word、Excel、图片格式
- 点击"查看"在新标签页打开文件
- 经理/管理员可上传和删除

## 8. 参考表单

**路径**：左侧菜单 → 资源 → 参考表单

参考表单是系统预设的标准化查验模板库，按专业分类：

| 分类 | 编码 | 内容范围 |
|------|------|----------|
| A. 设备用房查验 | A001-Axxx | 配电室、水泵房、消防泵房、电梯机房等 |
| B. 公共部位查验 | B001-Bxxx | 大堂、走廊、楼梯间、外墙、屋面等 |
| C. 室内专有部分查验 | C001-Cxxx | 户内墙面、地面、门窗、水电设施等 |
| D. 资料与物品移交 | D001-Dxxx | 竣工图、设备资料、钥匙、工具等 |

### 使用方式

1. 浏览找到需要的模板
2. 点击"查看"预览模板内容
3. 点击"保存为我的表单"创建个人副本
4. 之后可在"我的表单"中编辑和使用

### 管理员功能

- 新建参考表单：创建新的系统模板
- 编辑：修改现有模板的编号、标题、类别、条目

## 9. 打印

### 9.1 入口

有两种方式进入打印：
- 我的表单页面，点击表单卡片的"打印"按钮
- PrintSelect 页面，可选择多个表单和记录批量打印

### 9.2 打印预览

打印预览页面展示 A4 纸张格式的最终效果：

空白表包含：
- 项目名称标题
- 表单名称副标题
- 项目信息栏（名称、日期___、位置___、查验人___）
- 检查项目表（序号、检查项目、检查标准、查验结果）
- 查验意见栏
- 签名栏（查验人签字、日期）

已填记录额外包含：
- 每项的查验结果（合格/不合格/免检）
- 不合格项的问题描述
- 查验意见内容
- 照片附件（如有）

### 9.3 分页规则

- 第一页最多显示 22 行数据
- 第二页最多显示 26 行数据
- 超出的数据自动分到后续页面
- 照片单独分页展示

### 9.4 打印操作

点击"浏览器打印"按钮，使用系统打印对话框：
- 纸张尺寸：A4
- 边距：默认
- 建议勾选"背景图形"以确保表格边框正常显示
- 按 Ctrl+P（Mac: Cmd+P）也可直接打印

## 10. 仪表盘

**路径**：左侧菜单 → 账户 → 仪表盘（经理/管理员专属）

仪表盘提供项目的统计分析和数据概览。

### 10.1 时间范围

- 起始月份和截止月份选择器
- 快捷按钮：近 3 月 / 近 6 月 / 近 1 年

### 10.2 概览卡片

| 指标 | 说明 |
|------|------|
| 总记录数 | 时间范围内的查验记录总量 |
| 不合格项 | 所有记录中标记为"不合格"的条目数 |
| 已完成 | 状态为"已完成"的记录数 |
| 待处理 | 总记录数 - 已完成数 |

### 10.3 月度趋势图

以横向柱状图展示每月已完成和待处理的记录数量。

### 10.4 按模板统计

每个表单模板的统计：总数、已完成数、完成率百分比。

### 10.5 按人员统计

每位成员的统计：总数、已完成数、完成率百分比。

### 10.6 最近提交记录

显示最近提交的查验记录，点击可查看详情。

## 11. 项目配置

**路径**：左侧菜单 → 账户 → 项目配置（经理/管理员专属）

### 11.1 项目信息

编辑项目的基本信息：

- 项目名称、类型（工业/商业/住宅/其他）
- 地址、占地面积、建筑面积
- 交接日期、开发商
- 物业经理姓名、联系电话、联系邮箱

### 11.2 楼栋管理

- 添加楼栋：输入名称后回车或点击添加
- 删除楼栋：点击楼栋卡片的删除按钮
- 每个楼栋卡片显示包含的房源数量

### 11.3 房源管理

- 筛选：可按楼栋筛选
- 添加房源：选择楼栋、输入房号、面积
- 删除房源：点击房源卡片的删除按钮

### 11.4 成员管理

管理项目成员及其角色。

#### 添加成员

1. 点击"+ 添加成员"按钮
2. 系统自动生成员工编号（SG + 3位数字）
3. 输入姓名，选择角色（基层员工/物业经理/系统管理员）
4. 点击添加
5. 添加成功后弹出凭据窗口，显示用户名和自动生成的密码。请立即截图或复制保存，关闭后无法找回。

#### 成员列表

| 列 | 说明 |
|----|------|
| 编号 | 系统自动分配的用户名（SG格式） |
| 姓名 | 成员显示名称 |
| 角色 | 管理员 / 经理 / 员工 |
| 操作 | 移除成员、重置密码 |

#### 重置密码

1. 点击成员的"重置密码"按钮
2. 系统生成新的 6 位数字密码
3. 确认后弹出新密码，需告知用户本人

> 注意：系统不提供用户自行修改密码的功能（管理员重置除外），成员密码由管理员统一管理。

## 附录 A：常见问题

### Q: 提交后还能修改记录吗？
A: 不能。提交是不可逆操作，提交后任何人都无法修改或删除该记录。请在提交前仔细核对。

### Q: 为什么打印出来只有一页？
A: 检查项目总数超过 22 项时会自动分页。如果仍未分页，请尝试清除浏览器缓存后刷新。

### Q: 照片上传失败怎么办？
A: 检查照片大小是否超过 10MB，格式是否为常见图片格式（JPG/PNG/WebP）。网络不稳定也可能导致失败，请重试。

### Q: 忘记密码怎么办？
A: 联系你的物业经理或系统管理员重置密码。管理员在"项目配置 → 成员管理"中可以重置任何成员的密码。

### Q: 如何查看他人的记录？
A: 经理和管理员可以查看所有人的记录。基层员工只能查看自己创建的记录。

### Q: 侧边栏点击跳到了错误的页面？
A: 可能是浏览器缓存问题，按 Cmd+Shift+R（Mac）或 Ctrl+Shift+R（Windows）强制刷新。
