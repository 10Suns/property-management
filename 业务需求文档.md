# 瑞界物业 · 承接查验系统 — 业务需求文档

> 版本: V2.0 | 日期: 2026-05-21 | 基于会议讨论确认

---

## 1. 系统定位

两个核心功能：
1. **提供表单** — 员工选择模板、删减调整后打印空白表，外出手工记录
2. **存储电子记录** — 员工回传录入、管理员和物业经理在线查看

---

## 2. 公司信息

- 公司名称：**瑞界物业管理有限公司**
- 打印抬头：**瑞界物业 · 物业承接查验记录表**
- 登录用户名格式：**SG001**（公司缩写 `SG` + 3位数字工号）

## 4. 角色业务流程

### 4.1 基层员工完整流程

```
┌─────────────────────────────────────────────────┐
│                  办 公 室 （线上）                 │
│                                                   │
│  1. 登录系统（工号 SG001 + 密码）                  │
│  2. 进入项目 → 看到「参考表单」和「我的表单」两个标签 │
│  3. 在「参考表单」中选择被授权的模板                 │
│  4. 点击模板进入编辑页                              │
│  5. 批量编辑：删减不需要的项、调整标准、添加自定义项  │
│  6. 点击「保存为我的表单」                          │
│  7. 切换到「我的表单」标签                          │
│  8. 找到刚保存的表单 → 点击「打印空白表」            │
│  9. 打印纸质表，携带外出                            │
└──────────────────┬────────────────────────────────┘
                   │
                   ▼  带着纸和笔去现场
┌─────────────────────────────────────────────────┐
│                  现 场 （线下）                    │
│                                                   │
│  逐项检查：                                       │
│    - 合格 → 打勾                                  │
│    - 不合格 → 记录问题描述                         │
│    - 免检 → 标注                                  │
│  手机拍摄问题照片                                  │
│  手工签字、写日期                                  │
└──────────────────┬────────────────────────────────┘
                   │
                   ▼  回到办公室
┌─────────────────────────────────────────────────┐
│                  办 公 室 （线上）                 │
│                                                   │
│  1. 登录系统 → 项目 → 「我的表单」                 │
│  2. 找到对应表单 → 点击「开始查验」                 │
│  3. 对照纸质表，逐项录入结果：                      │
│     - 点击「合格」/「不合格」/「免检」              │
│     - 不合格项 → 填写问题描述                      │
│     - 点击 📷 按钮上传现场照片（手机也可操作）      │
│  4. 填写「查验意见」                               │
│  5. 将状态改为「已完成」 → 自动保存                │
│  6. 经理/管理员即可在线查看                        │
└─────────────────────────────────────────────────┘
```

## 5. 表单编辑规则

### 5.1 从模板到我的表单

| 字段 | 是否可改 | 说明 |
|------|:------:|------|
| 模板编号（如 A1） | ❌ 锁定 | 系统自动带出 |
| 表单标题（如"变配电室查验单"） | ❌ 锁定 | 保持模板原标题 |
| 大类别（如"设备机房查验"） | ❌ 锁定 | 分类归属不可变 |
| 检查项目名称 | ✔️ 可改 | 双击编辑 |
| 检查标准 | ✔️ 可改 | 双击编辑 |
| 检查项可删除 | ✔️ | 删除时不再逐个确认，统一在保存时提交 |
| 检查项可新增 | ✔️ | 点击"+ 添加条目" |
| 保存后表单命名 | 自动 | 格式：`原模板名 - 员工姓名` |

### 5.2 删除确认规则
- 编辑状态下点击「删除」：**直接删除，不弹确认框**
- 所有增删改操作：**统一在点击「保存为新表单」时提交**
- 取消编辑或离开页面：**所有修改丢弃**

## 6. 界面布局规则

### 6.1 项目详情页
- 顶部：项目名 + 基本信息
- 两个标签页：「参考表单」和「我的表单」
- 「参考表单」：显示该员工被授权的系统模板（按A/B/C/D分类）
- 「我的表单」：显示该员工自己创建的所有表单

### 6.2 表单编辑器
- 检查项目表格
- 批量编辑模式下显示「保存为我的表单」按钮
- 单条删除不弹窗，统一保存
- 编辑完成后点击「保存为我的表单」提交所有修改

### 6.3 打印入口
- **仅在「我的表单」中提供打印入口**
- 「参考表单」中不提供打印按钮
- 打印预览表头：**瑞界物业 · 物业承接查验记录表**

## 7. 打印格式规范

- 纸张：A4
- 抬头：
  ```
  瑞界物业 · 物业承接查验记录表
  A1  变配电室查验单
  ```
- 表头列宽：序号 4%、检查项目 9%、检查标准 40%、查验结果 47%
- 页脚：查验意见 + 签字区
- 照片另起一页

## 9. 移动端支持

- 系统采用响应式网页设计
- 手机端可通过浏览器访问（无需安装APP）
- 手机端功能：登录、查看表单、录入查验结果、上传照片
- 照片上传：支持手机相机直接拍照或从相册选择
