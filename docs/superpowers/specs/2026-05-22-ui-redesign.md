# UI 全面重构设计文档

日期：2026-05-22 | 状态：已确认

## 核心变更

1. 去除项目列表页，单项目系统，登录直达工作台
2. 全宽布局替代 max-width:800px，顶栏+侧边栏+内容区
3. 侧边栏 4 项：我的表单、检查记录、设备档案（工作台）+ 参考表单（资源）
4. 仪表盘增加月份范围选择器，支持月度巡检追踪
5. 简化权限，三个角色共用同一界面（功能按角色显示/隐藏）

## 页面清单

### 1. 登录页
- 420px 卡片居中
- 标题"物业承接查验" 28px 加粗
- 用户名+密码+登录按钮
- 登录后直接进入项目工作台（无项目列表）

### 2. 布局框架
- 顶栏：项目名称(左) + 仪表盘/配置 + 用户名(右)，蓝色背景
- 左侧 200px 侧边栏，分组导航
- 右侧内容区全宽自适应，背景 #f5f7fa

### 3. 侧边栏
- 工作台：我的表单、检查记录、设备档案
- 资源：参考表单（带 28 套计数徽标）
- 当前选中项蓝色高亮

### 4. 我的表单
- 顶部：标题 + "创建空白表单"按钮
- 引导说明文字
- 表单卡片：标题 + 编号标签 + 项数 + 日期
- 操作：打印空白表 / 编辑 / 删除
- 空状态引导去参考表单

### 5. 检查记录列表
- 筛选栏：按表单、楼栋、状态
- 记录卡片：表单名、人员、位置、日期、不合格数
- 状态标签：查验中(蓝) / 已完成(绿) / 待处理(红)
- 点击进入检查录入界面

### 6. 检查录入界面
- 位置卡片：楼栋+房源+具体位置
- 检查项目表格：序号 | 检查项目(28%) | 检查标准(28%) | 查验结果(200px) | 照片(52px)
- 合格/不合格/免检 圆角单选按钮
- 不合格时红色边框问题描述输入框
- 照片按钮：有照片时蓝色实心显示数量
- 底部：查验意见 textarea + 状态下拉

### 7. 参考表单
- 按 A/B/C/D 四类分组
- 双列网格展示模板
- 每行：编号标签 + 名称 + 项数 + "+ 创建"按钮
- 点击直接创建个人表单

### 8. 设备档案
- 筛选栏：分类 + 状态
- 设备卡片：名称 + 三列信息网格(分类/型号/编号/日期/位置/备注)
- 状态标签 + 编辑/删除按钮

### 9. 打印预览
- A4 纸模拟预览
- 工具栏：标题 + 开始打印 + 返回
- 列宽：序号 4% + 检查项目 9% + 检查标准 40% + 查验结果 47%
- 两种模式：空白表（结果列空）+ 已填写记录（含结果和照片）

### 10. 仪表盘
- 月份范围选择器：从 X 月到 Y 月，最长 1 年
- 快捷按钮：近 3 月 / 近 6 月 / 近 1 年
- 4 统计卡片：总记录、不合格、已完成、待处理
- 双列：按模板统计 + 按员工统计（含完成率）
- 月度趋势柱状图（蓝=完成，红=待处理）
- 最近记录列表

### 11. 配置
- 三个标签页：楼栋管理 / 房源管理 / 成员管理
- 楼栋：输入框+添加，列表含户数和删除
- 房源：选择楼栋，添加房号+面积
- 成员：添加姓名+角色，自动生成 SG 账号

## 技术要点

- 使用 CSS Grid 和 Flexbox，移除 max-width 限制
- 侧边栏固定 200px，使用 Vue Router 路由切换
- 仪表盘月份数据通过 API 查询参数传递
- 保留现有后端 API 不变，仅前端重构
