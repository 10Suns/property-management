## 数据关系图

```
projects (项目)
  ├── buildings (楼栋)
  │     └── houses (房源)
  ├── inspection_templates (参考模板)
  │     └── template_items (模板检查项)
  ├── user_forms (我的表单)
  │     └── user_form_items (表单检查项)
  ├── inspection_records (查验记录)
  │     ├── inspection_results (查验结果，逐项)
  │     └── inspection_photos (查验照片)
  └── equipment (设备)
        ├── maintenance_records (维护记录)
        └── equipment_manuals (说明书)

users (用户) → 上述所有表的 created_by / user_id
```

## "我的表单"核心流程涉及的表

1. **创建表单**：`user_forms` ← 从 `inspection_templates` + `template_items` 复制 → `user_form_items`
2. **打印空白表**：读 `user_forms` + `user_form_items`
3. **新增记录**：`inspection_records` ← 关联 `user_forms`
4. **录入结果**：`inspection_results`（每项一条） ← 关联 `inspection_records`
5. **上传照片**：`inspection_photos` ← 关联 `inspection_results`
6. **提交锁定**：`inspection_records.submitted = 1`
